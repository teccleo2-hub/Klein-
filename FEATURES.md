# FLEKO-X BOT - Features List

FLEKO-X BOT is a highly customizable WhatsApp bot with a wide range of features for automation, security, and group management.

## 🚀 Core Features
- **Prefix Support:** Customizable command prefix (default: `.`).
- **Mode:** Switch between `public` and `private` modes.
- **Always Online:** Keep the bot status as "Online" even when idle.
- **Session Management:** Unique Session ID for secure and persistent connection.
- **Health Checks:** Built-in API endpoints for monitoring bot status.

## 🛡️ Security & Anti-Features
- **Anti-Delete:** Detects and notifies when a message is deleted.
- **Anti-Edit:** Detects and notifies when a message is edited.
- **Anti-Call:** Automatically rejects incoming calls to the bot.
- **Auto-Block:** Automatically blocks users who violate bot rules (e.g., calling).
- **Anti-Sticker:** Options to restrict sticker usage (Kick/Warn).
- **Anti-Bug:** Protection against malicious "bug" messages.

## 🤖 Automation
- **Auto-Bio:** Automatically updates the bot's status/bio.
- **Auto-Read:** Automatically marks incoming messages as read.
- **Auto-Type:** Shows "typing..." status when processing commands.
- **Auto-Record:** Shows "recording..." status for audio/video messages.
- **Auto-View Status:** Automatically views contacts' status updates.
- **Auto-React Status:** Automatically reacts to status updates with emojis.

## 👥 Group Management
- **Welcome/Goodbye:** Customizable messages for new and departing members.
- **Admin Tools:** Kick, Promote, Demote, and Group Info commands.
- **Member List:** Quickly list all members in a group.
- **Restricted Mode:** Limit bot usage to specific group members or admins.

## 🎨 Branding & Customization
- **Custom Bot Name:** Set your own name for the bot.
- **Watermark:** Add a custom watermark to bot responses.
- **Sticker Pack:** Customize the author and pack name for generated stickers.
- **Menu Styles:** Multiple menu layouts to choose from.
- **Font Styles:** Support for different text fonts in bot messages.

## 📊 Monitoring & Info
- **Bot Info:** Detailed information about the bot's configuration and status.
- **Uptime:** Track how long the bot has been running.
- **Timezone:** Localized time reporting based on your region.
